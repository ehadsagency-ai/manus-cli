# Manus CLI Tests

This directory contains comprehensive test suites for Manus CLI features.

## Test Scripts

### v5.1 Features
- `test_v5.1_features.py` - Tests for Extended Thinking, Effort, Templates, Validation

### v5.2 Features
- `test_v5.2_features.py` - Tests for Cache, Context, Evaluation, Monitoring

## Running Tests

```bash
# Run all v5.1 tests
python3 tests/test_v5.1_features.py

# Run all v5.2 tests
python3 tests/test_v5.2_features.py
```
